# Python 示例程序

## 项目介绍

本项目是一个使用 [Flask](http://flask.pocoo.org) 框架编写的简单的Python WEB示例程序, 目录结构：

```
.
├── README.md
├── hello.py
├── runtime.txt
└── requirements.txt
```

pip install -r requirements.txt --allow-all-external


```
gunicorn hello:app --log-file - --access-logfile - --error-logfile -
```
